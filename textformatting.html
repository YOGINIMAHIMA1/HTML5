ALGORITHM

algorithm means "a set of rules to be followed in calculations or other problem solving operation "OR" a procedure to solve a set of mathematical problems in a finite number of steps that frequently by recursive functions.

Algorithm can be simple and complex depending on what you want to achieve.

TYPES OF ALGORITHMS

- Brute force algorithm- it is the simplest approach for a problem. it is the first algorithm that comes when we see a problem.
recursive algorithm-it is based on recursion.in this case, problem is divided into sub-parts and called the same function again and again. Backtracking algorithm-it basically build the solution by searching among all possible solution.
